# DISCORD QUORUM GOVERNANCE PRINCIPLES

## 1. Purpose
Define governance invariants for deterministic, quorum-controlled Discord administration.

## 2. Core Principles
1. Administrative authority is quorum-mediated.
2. Operational Discord actions are executed by GuildBot only.
3. Custody authority is externalized to Custody API.
4. Cryptographic determinism is centralized in Crypto Core.
5. Governance and custody boundaries must remain explicit and auditable.

## 3. Compliance
All automation must preserve Discord Terms of Service compliance.
